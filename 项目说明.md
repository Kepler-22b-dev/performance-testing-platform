# 性能测试平台

基于 Apache JMeter 的分布式性能测试平台，提供 Web 界面管理压测任务、分析结果、监控资源。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 🎯 压测执行 | 分布式 JMeter 压测，支持多 Slave 节点 |
| 📊 结果分析 | TPS/RT/错误率曲线、分布图、百分位分析 |
| 📱 移动端监控 | iOS (pymobiledevice3) + Android (SOLOX) 性能采集 |
| ⏱️ 定时调度 | 间隔执行、单次定时 |
| 🔔 告警通知 | 阈值检测 + Webhook 通知 |
| 📈 趋势分析 | 历史性能趋势、任务对比 |

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | HTML/CSS/JS + ECharts | 5.4.3 |
| 后端 | Python + FastAPI | 3.10+ |
| 消息队列 | Redis | 6.0+ |
| 压测引擎 | Apache JMeter | 5.6.3 |
| iOS监控 | pymobiledevice3 | 9.x |
| Android监控 | SOLOX | 2.9.x |

## 快速开始

### 环境要求

| 组件 | 版本 | 说明 |
|------|------|------|
| Python | 3.10+ | Manager 和 Agent |
| Java | 8+ | JMeter 需要 |
| Redis | 6.0+ | 消息队列 |

### 一键部署

```bash
git clone https://github.com/Kepler-22b-dev/performance-testing-platform.git
cd performance-testing-platform
bash deploy.sh
```

### 手动部署

```bash
pip install -r manager/requirements.txt
pip install -r agent/requirements.txt
python3 -m manager.main &
python3 -m agent.main &
```

访问 http://localhost:8000

### CLI 工具

```bash
python3 cli.py --help
```
