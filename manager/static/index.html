功能特性

✓
分布式压测
支持多台施压节点并行执行，水平扩展压力
⚙
脚本在线编辑
Web 端直接编辑 JMeter 脚本，支持保存、格式化、一键执行
📈
实时监控
CPU/内存/网络实时监控，JMeter 进程资源追踪
📊
结果分析
TPS/响应时间/错误率/百分位分布，多维度可视化图表
📁
数据驱动
CSV 数据文件管理，全局变量配置，支持参数化压测
⏰
定时调度
支持定时/周期性执行压测任务，自动化测试
🔔
通知系统
任务完成后通过 Webhook 自动通知
📈
任务对比
对比两次压测结果，直观查看性能变化

快速开始

1. 一键部署
bash deploy.sh
2. 访问平台
http://localhost:8000
3. 上传脚本
进入「脚本管理」上传 .jmx 文件或新建脚本
4. 执行压测
进入「创建任务」配置参数并启动压测
5. 查看结果
在「任务管理」查看实时进度和分析报告
6. API 文档
访问 /docs 查看完整 API 文档
