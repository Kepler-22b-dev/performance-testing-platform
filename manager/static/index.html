快速开始

1. 一键部署
bash deploy.sh
2. 访问平台
http://localhost:8000
3. 上传脚本
进入「脚本管理」上传 .jmx 文件或新建脚本
4. 执行压测
进入「创建任务」配置参数并启动压测
5. 查看结果
在「任务管理」查看实时进度和分析报告
6. API 文档
访问 /docs 查看完整 API 文档

新手启动流程
1选择脚本上传或搜索 JMX 脚本，确认脚本和 CSV 数据是否匹配。
2配置场景选择冒烟、负载、压力等场景，系统会带出并发和时长。
3确认资源选择压力机，JVM 会按场景自动推荐，可按机器内存微调。
4启动查看确认右侧摘要无误后启动，在任务管理查看进度和报告。
